# Backend Laravel

## Struttura progetto
- Moduli principali: gestione precontrattuali, dashboard, generazione PDF
- Migrazioni e seeders per database
- API REST documentate in `api.md`

## Comandi principali
- `php artisan serve` per avviare il backend
- `php artisan migrate:fresh --seed`
